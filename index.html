uBlock

Requirements: add extention uBlock


1. Download uBlock origin
2. Click the extention and click the cog icon on the bottom right
3. Scroll to the bottom and enable "I am an advanced user" then, click that cog icon
4. Scroll to the bottom and add "userResourcesLocation https://raw.githubusercontent.com/Kitis2cool/shortcut/refs/heads/main/urun.js" without the quotation marks you might have to delete the old userResourcesLocation. Now apply changes
5. Go back the the first page where you enabled the check box and in the "My Filters" tab add "*##+js(urun.js)" without the quotations marks and apply changes
6. Go to any website (refresh old ones to update them) Testing site and use the keybind, "Crtl + Shift + ~" the "~" should be above the tab key.
7. To add code click the plus icon and add your code add a name and save it.
8. To use devtools click "<>" icon and at the bottom right there should be a button to click
9. DISCLAMER: This will not work on some sites.
